# Code of Conduct

## Our Commitment

We are committed to fostering a welcoming, safe, and respectful environment for everyone contributing to **Extra Wallet**, regardless of background, experience level, or personal identity.
This project values integrity, transparency, and collaboration.

## Expected Behavior

Contributors are expected to:

* Treat everyone with respect and professionalism.
* Communicate clearly, calmly, and constructively.
* Provide helpful feedback with the goal of improving the project.
* Use inclusive language and avoid personal attacks.
* Follow project guidelines and respect maintainersâ€™ decisions.

## Unacceptable Behavior

The following actions will not be tolerated:

* Harassment, discrimination, or hateful conduct.
* Personal insults, trolling, or intentionally disruptive behavior.
* Sharing private information without permission.
* Any behavior that threatens project security, maintainability, or contributor safety.

## Reporting Issues

If you observe or experience violations of this Code of Conduct:

* Please report them confidentially to the project maintainer.
* Include context and any supporting information so the issue can be reviewed fairly.

We take reports seriously and will respond promptly.

## Enforcement

Project maintainers may take any action deemed appropriate, including:

* Requesting behavior changes
* Temporary or permanent removal from discussions and contributions
* Reverting, blocking, or rejecting contributions

Decisions will prioritize project health and contributor safety.

## Acknowledgment

This Code of Conduct is inspired by common open-source community standards, including the Contributor Covenant.
